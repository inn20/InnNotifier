## InnNotifier
---
##### 插件截图：
![image](https://user-gold-cdn.xitu.io/2020/6/26/172ef4b460ab9b37?w=1237&h=803&f=png&s=41225)

#### typecho通知插件，支持微信、QQ、BearyChat

##### 支持的推送方式：
- Server酱(微信推送)
- BearyChat
- Qmsg酱(QQ推送)

##### 推送类型：
- 评论通知 
- 注册通知
